# generator-customer

  该模块是一个脚手架生成器，用来输出常用的开发目录结构。该模块中内置了两个生成器：
  app : 默认。用于开发后端模板使用velocity的前端项目。
  react : 用于开发一个前后端同构的react应用。

# install and use

    npm install generator-customer -g  
    mkdir test 
    cd test
    yo customer

  or:
  
    npm install generator-customer -g  
    mkdir test 
    cd test
    yo customer:react
